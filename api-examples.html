# API 使用示例

展示 ol-base-components 相关脚本的典型调用方式。

# 生成 API 模块

npx api http://your-api-domain ./src/api/modules

# 生成 Swagger 路径映射

npx run http://your-api-domain ./src/api/swagger.js

# 一键生成（init）

npx init http://your-api-domain
